# Labyrinth Game 

Консольная игра-лабиринт: войдите в лабиринт, полный секретов и смертельных ловушек, и попробуйте открыть сундук с сокровищем!

## Установка и запуск

Требования: python3.13+ и git.

1) Клонируйте репозиторий и перейдите в корень проекта.

2) Установите Poetry (если не установлен):
```bash
python3 -m pip install --user poetry
```

3) Установите зависимости:
```bash
poetry install
```

4) Запустите игру:
```bash
poetry run project
```
## Альтернатива через Makefile

В корне репозитория:
```bash
make install
make project
```
